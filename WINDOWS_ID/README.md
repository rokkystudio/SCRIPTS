# README

PowerShell-скрипт выводит системные идентификаторы Windows из реестра:

- `MachineGuid` — уникальный идентификатор установки Windows.
- `ComputerName` — текущее имя компьютера в системе.
- `Hostname` — сетевое имя хоста, которое используется стеком TCP/IP.
- `SID` профилей — уникальные идентификаторы учетных записей и пути к их профилям.
- `DigitalProductId` — признак наличия в реестре данных, связанных с идентификатором продукта Windows.
